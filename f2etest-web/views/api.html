使用F2etest中的浏览器打开指定URL

  • DEMO地址：http://<%=host%>/openapp?id=ie8&url=http://www.alibaba.com/
  • 此接口支持以下参数：
    1. id: 浏览器ID，例如：ie6, ie7, ie8, ie9, ie10, ie11, chrome, firefox
    2. url: 需要打开的目标URL地址
    3. noproxy: 是否禁用代理，设置1为禁用，禁用后即访问线上
    4. proxyurl: 代理服务器pac地址，可利用代理服务器实现自定义hosts绑定功能
